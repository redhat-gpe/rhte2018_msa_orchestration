Microservice Orchestration using PAM 7

This multi-product lab guides you through the creation of a modern container native application where inter-service interaction is orchestrated and the communication is asynchroneous.

The architectural patterns utilized in this lab are increasingly considered best practices in many circumstances. The lab is deployed on Red Hat OpenShift Container Platform and includes the following components:
  • Red Hat Process Automation Server
  • Red Hat Enmasse
  • Red Hat OpenShift Application Runtimes
  • Jaeger
